# Study Sprint Planner

## Overview

Study Sprint Planner is a React web application created for the CSE 310 Web Apps module. The purpose of the app is to help a student plan a two-week learning sprint, track study tasks, monitor progress toward the 20-hour sprint goal, and organize work by category.

The app lets the user add, edit, delete, complete, search, and filter sprint tasks. It also calculates planned hours, completed hours, sprint progress, open task count, and hours by category. Task data is saved in browser local storage so the sprint plan remains available after the page refreshes.

## Running the Program

Install dependencies:

```bash
npm install
```

Start the local development server:

```bash
npm run dev
```

Build the production version:

```bash
npm run build
```
